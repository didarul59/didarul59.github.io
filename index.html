MnO Wannier90 Tutorial: Complete Workflow Analysis

1. Introduction

This document summarizes the successful Wannier90 calculation for MnO (Manganese Oxide), an antiferromagnetic insulator with rock-salt structure. Both up and down spin channels ran successfully with identical input files, providing a reference for more complex materials like FeO.

2. Workflow Steps

Step 1: Generate k-points for NSCF
~/q-e-qe-7.3/external/wannier90/utility/kmesh.pl 4 4 4 >> MnO_nscf.in
Step 2: Run NSCF Calculation
mpirun -np 4 pw.x -i MnO_nscf.in > MnO_nscf.out
Step 3: Generate k-points for Wannier90
~/q-e-qe-7.3/external/wannier90/utility/kmesh.pl 4 4 4 wan >> MnO_up.win ~/q-e-qe-7.3/external/wannier90/utility/kmesh.pl 4 4 4 wan >> MnO_down.win
Step 4: Wannier90 Pre-processing
wannier90.x -pp MnO_up wannier90.x -pp MnO_down
Step 5: Run pw2wannier90
pw2wannier90.x < pw2wan_MnO_up.in > pw2wan_MnO_up.out pw2wannier90.x < pw2wan_MnO_down.in > pw2wan_MnO_down.out
Step 6: Run Wannier90
wannier90.x MnO_up wannier90.x MnO_down
Step 7: Verify Success
# Check for convergence grep "Final State" MnO_up.wout grep "Spread" MnO_up.wout # Plot bands gnuplot MnO_up_band.gnu

3. Understanding nbnd Calculation

3.1 Electron Counting for MnO
Atom Configuration Valence Electrons Count Total
Mn [Ar] 4s² 3d⁵ 7 2 14
O [He] 2s² 2p⁴ 6 2 12
Total 26
3.2 Band Calculation Logic
Total valence electrons = 26
Spin-polarized (nspin=2) → each band holds 1 electron
Valence bands needed = 26 bands
Extra empty bands for accuracy = 10 bands
Total nbnd = 36
3.3 Relationship with Wannier90 Parameters
nbnd (36) ≥ num_bands (26) ≥ num_wann (18) + exclude_bands (10)
  • nbnd = 36: What pw.x calculates (includes empty bands)
  • num_bands = 26: What Wannier90 reads from DFT (valence bands)
  • exclude_bands : 1-10: Removes core states
  • num_wann = 18: Final Wannier functions

5. Key Insights

5.1 Identical .win Files for Both Channels
  • MnO's symmetric band structure allows the same parameters for up and down
  • Both channels ran successfully without modification
5.2 Relationship Between nbnd and Wannier Parameters
Parameter Value Meaning
nbnd 36 Total bands calculated by pw.x
num_bands 26 Bands used by Wannier90
exclude_bands 1-10 Core states excluded
num_wann 18 Target Wannier functions
Constraint: nbnd ≥ num_bands ≥ num_wann + excluded_bands
5.3 Projections Count
Mn:s;d → 1s + 5d = 6 orbitals per Mn × 2 Mn = 12
O:p → 3 orbitals per O × 2 O = 6
Total = 18 Wannier functions
5.4 Energy Windows
  • Outer window (5-35 eV): Contains all relevant bands
  • Frozen window (5-18 eV): Inner window for disentanglement
  • Works because MnO has a clean band gap
5.5 Why FeO is Different

Unlike MnO, FeO shows:

  • Asymmetric band structures between up and down channels
  • Deep core states in down channel (-72 eV to -38 eV)
  • Different total band counts (72 vs 60)
  • Requires channel-specific parameters

6. Troubleshooting Guide

6.1 Common Errors and Solutions
Error Cause Solution
mismatch in .eig file num_bands doesn't match DFT Check actual bands: awk '{print $1}' FeO_up.eig | sort -nu | tail -1
fewer states in frozen window Not enough bands in energy range Widen energy windows or use frozen_bands
more states in frozen window Too many bands in range Narrow window or use exact frozen_bands
k-point mismatch K-points differ between files Use kmesh.pl for consistency
6.2 Diagnostic Commands
# Check total bands in .eig file awk '{print $1}' MnO_up.eig | sort -nu | tail -1 # Check k-point count awk '{print $2}' MnO_up.eig | sort -nu | tail -1 # Check energy range awk '{print $3}' MnO_up.eig | sort -n | head -1 awk '{print $3}' MnO_up.eig | sort -n | tail -1 # Check specific bands at first k-point awk '$2==1 && $1>=40 && $1<=50 {print}' MnO_up.eig
