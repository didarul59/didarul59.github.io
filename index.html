MnO Wannier90 Tutorial: Complete Workflow Analysis

This tutorial summarizes the successful Wannier90 calculation for MnO (Manganese Oxide), an antiferromagnetic insulator with rock-salt structure. Both up and down spin channels ran successfully with identical input files, providing a reference for more complex materials like FeO.

1. Introduction

MnO is a classic example of a strongly correlated transition metal oxide. Using Wannier90, we can construct maximally localized Wannier functions (MLWFs) to analyze its electronic structure with high precision.

2. Understanding nbnd Calculation

Atom Configuration Valence Electrons Count Total
Mn [Ar] 4s² 3d⁵ 7 2 14
O [He] 2s² 2p⁴ 6 2 12
Total 26
Total valence electrons = 26
For spin-polarized (nspin=2) → 26 bands needed for valence states.

3. NSCF Input Highlights

MnO_nscf.in snippet
&system ibrav = 0, celldm(1) = 7.88, nat = 4, ntyp = 3, ecutwfc = 50.0, nspin = 2, nbnd = 36 ! 26 valence + 10 empty /

4. Projections & Wannier Parameters

We target 18 Wannier functions: 12 from Mn (s + d) and 6 from O (p).

Begin Projections Mn:s;d # 2 Mn × (1s + 5d) = 12 O:p # 2 O × 3p = 6 End Projections num_wann = 18 num_bands = 26 exclude_bands : 1-10

5. Workflow Steps

Follow these steps to complete the calculation:

# Step 1: Run NSCF mpirun -np 4 pw.x -i MnO_nscf.in > MnO_nscf.out # Step 2: Wannier90 Pre-processing wannier90.x -pp MnO_up # Step 3: Run pw2wannier90 pw2wannier90.x < pw2wan_MnO_up.in > pw2wan_MnO_up.out # Step 4: Run Wannier90 wannier90.x MnO_up

Conclusion

The identical behavior of up and down channels in MnO simplifies the workflow. For materials like FeO, you must account for asymmetry by checking the .eig files for each channel separately.
